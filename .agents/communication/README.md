# Multi-Agent Workflow Communications Log

This directory contains authoritative task specifications, execution results, code reviews, and verification logs for multi-agent orchestration between Codex (Planner & Reviewer) and Agy (Implementer & Test Runner).

## Structure:
- `task_specifications/`: Formal task prompts with acceptance criteria and test requirements.
- `execution_logs/`: Test run output, proof receipts, and implementation evidence.
- `reviews/`: Formal code review decisions (APPROVE / REQUEST_CHANGES / ESCALATE).
